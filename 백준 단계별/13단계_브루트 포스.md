## 2798번. 블랙잭

```python
# 블랙잭
n,m=map(int,input().split())
can=list(map(int,input().split()))
v=[]
for i in range(0,n-2):
    for j in range(i+1,n-1):
        for k in range(j+1,n):
            v.append(can[i]+can[j]+can[k])
print(max([x if x<=m else 0 for x in v]))
```



## 2231번. 분해합

```python
# 분해합
n=int(input())
v=[]
for x in range(1,n):
    sum=0
    y=x
    while y>=1:
        sum+=y%10
        y=y//10
    if sum+x==n:
        v.append(x)
    else:
        pass
if v==[]:
    print(0)
else:
    print(min(v))
```



## 7568번. 덩치

```python
def cnt(x, arr):
    count1 = 0
    for i in arr:
        if i[0] > x[0] and i[1] > x[1]:
            count1 += 1
    return count1+1


N = int(input())
arr = [list(map(int, input().split())) for _ in range(N)]
for i in arr:
    print(cnt(i, arr), end=' ')
```

## 1436번. 영화감독 숌

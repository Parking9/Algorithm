##  4673번 [셀프 넘버](https://www.acmicpc.net/problem/4673)


```python
def d(n):
    a=n
    v=[]
    while a >0:
        v.append(a % 10)
        a=a//10
    return(n+sum(v))
v1=[]
for i in range(0,10000):
    v1.append(d(i))
for i in list(range(1,10000)):
    if i not in v1:
        print(i, end='\n')
```
